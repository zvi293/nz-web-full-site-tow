# NZ-web Full Site

אתר React/Vite מלא ומוכן להרצה.

## התקנה והרצה

```bash
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```

## קבצים מרכזיים

- `src/App.jsx` — כל האתר והקומפוננטות
- `src/main.jsx` — נקודת כניסה של React
- `src/styles.css` — CSS גלובלי
- `index.html` — HTML ראשי
- `package.json` — תלותים וסקריפטים

## תיקון הבאנר הנגלל הקיים

בוצע תיקון לבאנר הנגלל שכבר היה קיים באתר:
- לא נוסף באנר חדש.
- הוסר שימוש ב־`text-stroke` כהה שגרם לטקסט להיבלע ברקע.
- נוספו צבע אמיתי, קונטרסט, Glow, מסגרת מדורגת וריווח תקין.
- הבאנר הוגבל לרוחב מתאים ונראה טוב גם במובייל.
- הבאנר נשאר במיקום הקיים מתחת לאזור ההדר/הירו.
